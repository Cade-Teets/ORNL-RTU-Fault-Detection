# ORNL-RTU-Fault-Detection
Physics-based Fault Detection for commercial Rooftop Units (RTUs) using the ORNL dataset. Focuses on detecting stuck outdoor air dampers using physics-informed Outside Air Fraction (OAF) calculations. Features energy signature visualizations and seasonal sensitivity testing to validate fault signatures.
